RF Object Pool
--------------
Game object pooling manager.  Offers the following functionality:

* 1). Spawning / Pooling of different game objects
* 2). Reparenting despawned objects, if requested (keep that scene hierarchy clean!)
* 3). Editor Hookup or/and API calls to initiate object pools
* 4). Object life-time management
* 5). Per-Scene or Global setup of pool (DontDestroyOnLoad)
* 6). Mass destruction / despawn
